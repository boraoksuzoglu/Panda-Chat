## Installation & Setup
### Install Repository
```git
git clone https://github.com/boraoksuzoglu/Panda-Chat
```

### Install Node Modules
```git
npm install
```

### Run Project
```git
npm start
```

### Finish
Now your project is running on **https://localhost:3000**
